Go иҜӯиЁҖж•ҷзЁӢзі»еҲ—гҒ®д»Ӣз»Қе’ҢзҺҜеўғе®үиЈ…

еҸ‘иЎЁдәҺ2023-03-19|зҙ«еҚҮзҝ»иҜ‘и®ЎеҲ’GolangGo иҜӯиЁҖзі»еҲ—ж•ҷзЁӢ
|еӯ—ж•°жҖ»и®Ў:1.2k|йҳ…иҜ»ж—¶й•ҝ:3еҲҶй’ҹ|йҳ…иҜ»йҮҸ:

иҝҷжҳҜжҲ‘д»¬ Golang ж•ҷзЁӢзі»еҲ—дёӯзҡ„з¬¬дёҖзҜҮж•ҷзЁӢгҖӮжң¬ж•ҷзЁӢд»Ӣз»ҚдәҶ GoпјҢе№¶и®Ёи®әдәҶйҖүжӢ© Go иҖҢдёҚжҳҜе…¶д»–зј–зЁӢиҜӯиЁҖзҡ„еҺҹеӣ гҖӮжҲ‘д»¬иҝҳе°ҶеӯҰд№ еҰӮдҪ•еңЁ Mac OSпјҢWindows е’Ң Linux дёӯе®үиЈ… GoгҖӮ

д»Ӣз»Қ

Go д№ҹиў«з§°дёә GolangпјҢжҳҜ Google ејҖеҸ‘зҡ„дёҖз§ҚејҖжәҗгҖҒзј–иҜ‘е’ҢйқҷжҖҒзұ»еһӢзҡ„зј–зЁӢиҜӯиЁҖгҖӮGo еҲӣдҪңиғҢеҗҺзҡ„е…ій”®дәәзү©жҳҜ Rob PikeпјҢKen Thompson е’ҢRobert GriesemerгҖӮGo дәҺ 2009 е№ҙ 11 жңҲе…¬ејҖеҸ‘еёғгҖӮ

Go жҳҜдёҖз§ҚйҖҡз”Ёзҡ„зј–зЁӢиҜӯиЁҖпјҢиҜӯжі•з®ҖеҚ•е№¶е…·жңүејәеӨ§зҡ„ж ҮеҮҶеә“дҪңдёәеҗҺзӣҫгҖӮGo еӨ§ж”ҫејӮеҪ©зҡ„е…ій”®йўҶеҹҹд№ӢдёҖжҳҜеҲӣе»әй«ҳеҸҜз”ЁжҖ§е’ҢеҸҜдјёзј©жҖ§зҡ„ Web еә”з”ЁзЁӢеәҸгҖӮGo иҝҳеҸҜд»Ҙз”ЁдәҺеҲӣе»әе‘Ҫд»ӨиЎҢеә”з”ЁзЁӢеәҸгҖҒжЎҢйқўеә”з”ЁзЁӢеәҸз”ҡиҮіз§»еҠЁеә”з”ЁзЁӢеәҸгҖӮ

Go зҡ„дјҳеҠҝ

еҪ“еӨ§йҮҸе…¶д»–иҜӯиЁҖпјҲдҫӢеҰӮ pythonпјҢrubyпјҢnodejs вҖҰпјүеҸҜд»Ҙе®һзҺ°зӣёеҗҢйңҖжұӮж—¶пјҢдёәд»Җд№ҲйҖүжӢ© Go дҪңдёәжңҚеҠЎеҷЁз«Ҝзј–зЁӢиҜӯиЁҖгҖӮ

иҝҷйҮҢжҳҜжҲ‘йҖүжӢ© Go ж—¶еҸ‘зҺ°зҡ„дёҖдәӣдјҳзӮ№гҖӮ

з®ҖеҚ•зҡ„иҜӯжі•

иҜӯжі•з®ҖеҚ•жҳҺдәҶпјҢе№¶дё”иҜҘиҜӯиЁҖдёҚдјҡеӣ дёҚеҝ…иҰҒзҡ„еҠҹиғҪиҖҢжҳҫеҫ—иҮғиӮҝгҖӮиҝҷдҪҝеҫ—зј–еҶҷеҸҜиҜ»жҖ§е’ҢеҸҜз»ҙжҠӨжҖ§зҡ„д»Јз ҒеҸҳеҫ—е®№жҳ“гҖӮ

жҳ“дәҺзј–еҶҷе№¶еҸ‘зЁӢеәҸ

е№¶еҸ‘жҳҜиҜӯиЁҖзҡ„еӣәжңүйғЁеҲҶгҖӮзӣҙжҺҘзҡ„еҪұе“ҚжҳҜзј–еҶҷеӨҡзәҝзЁӢзЁӢеәҸеҸҳеҫ—з®Җзӣҙе°ҸиҸңдёҖзўҹгҖӮиҝҷжҳҜйҖҡиҝҮ Goroutines е’Ңз®ЎйҒ“е®һзҺ°зҡ„пјҢжҲ‘д»¬е°ҶеңЁжҺҘдёӢжқҘзҡ„ж•ҷзЁӢдёӯеҜ№е…¶иҝӣиЎҢи®Ёи®әгҖӮ

зј–иҜ‘иҜӯиЁҖ

GoжҳҜдёҖз§Қзј–иҜ‘иҜӯиЁҖгҖӮжәҗд»Јз Ғиў«зј–иҜ‘дёәжң¬ең°дәҢиҝӣеҲ¶ж–Үд»¶гҖӮ и§ЈйҮҠиҜӯиЁҖпјҲдҫӢеҰӮпјҢnodejs дёӯдҪҝз”Ёзҡ„ JavaScript е’Ң Pythonпјүзјәе°‘жӯӨеҠҹиғҪгҖӮ

еҝ«йҖҹзј–иҜ‘

Goзј–иҜ‘еҷЁйқһеёёзҘһеҘҮпјҢе№¶дё”д»ҺдёҖејҖе§Ӣе°ұиў«и®ҫи®Ўдёәеҝ«йҖҹзҡ„гҖӮ

йқҷжҖҒй“ҫжҺҘ

Go зј–иҜ‘еҷЁж”ҜжҢҒйқҷжҖҒй“ҫжҺҘгҖӮж•ҙдёӘ Go йЎ№зӣ®еҸҜд»ҘйқҷжҖҒй“ҫжҺҘеҲ°дёҖдёӘеӨ§еһӢзҡ„дәҢиҝӣеҲ¶ж–Үд»¶дёӯпјҢе№¶дё”еҸҜд»ҘиҪ»жқҫйғЁзҪІеңЁдә‘жңҚеҠЎеҷЁдёӯпјҢиҖҢдёҚеҝ…жӢ…еҝғдҫқиө–е…ізі»гҖӮ

Go е·Ҙе…·й“ҫ

е·Ҙе…·еңЁ Go дёӯеҖјеҫ—зү№еҲ«жҸҗеҸҠгҖӮGo йҷ„еёҰдәҶеҠҹиғҪејәеӨ§зҡ„е·Ҙе…·пјҢеҸҜд»Ҙеё®еҠ©ејҖеҸ‘дәәе‘ҳзј–еҶҷжӣҙеҘҪзҡ„д»Јз ҒгҖӮеёёз”Ёзҡ„е·Ҙе…·жңүпјҡ

  • gofmtпјҡgofmt жҳҜз”ЁдәҺиҮӘеҠЁж јејҸеҢ– go жәҗд»Јз Ғзҡ„гҖӮе®ғдҪҝз”ЁеҲ¶иЎЁз¬ҰиҝӣиЎҢзј©иҝӣпјҢе№¶дҪҝз”Ёз©әж јиҝӣиЎҢеҜ№йҪҗгҖӮ
  • vetпјҡvet еҲҶжһҗ go жәҗд»Јз Ғе№¶жҠҘе‘ҠеҸҜиғҪзҡ„еҸҜз–‘д»Јз ҒгҖӮvet жҠҘе‘Ҡзҡ„жүҖжңүеҶ…е®№йғҪдёҚжҳҜзңҹжӯЈзҡ„й—®йўҳпјҢдҪҶе…·жңүжҚ•иҺ·зј–иҜ‘еҷЁжңӘжҠҘе‘Ҡзҡ„й”ҷиҜҜзҡ„иғҪеҠӣпјҢдҫӢеҰӮдҪҝз”Ё Printf ж—¶ж јејҸиҜҙжҳҺз¬ҰдёҚжӯЈзЎ®гҖӮ
  • golintпјҡgolint з”ЁдәҺиҜҶеҲ«д»Јз Ғдёӯзҡ„ж ·ејҸй—®йўҳгҖӮ

еһғеңҫеӣһж”¶

Go дҪҝз”Ёеһғеңҫеӣһж”¶пјҢеӣ жӯӨеҶ…еӯҳз®ЎзҗҶеҮ д№ҺдјҡиҮӘеҠЁжү§иЎҢпјҢејҖеҸ‘дәәе‘ҳж— йңҖжӢ…еҝғеҶ…еӯҳз®ЎзҗҶй—®йўҳгҖӮ иҝҷд№ҹжңүеҠ©дәҺиҪ»жқҫең°зј–еҶҷе№¶еҸ‘зЁӢеәҸгҖӮ

з®ҖеҚ•иҜӯиЁҖи§„иҢғ

иҜӯиЁҖи§„иҢғйқһеёёз®ҖеҚ•гҖӮ ж•ҙдёӘи§„иҢғеҸӘжңүдёҖдёӘйЎөйқўпјҢдҪ з”ҡиҮіеҸҜд»ҘдҪҝз”Ёе®ғжқҘзј–еҶҷиҮӘе·ұзҡ„зј–иҜ‘еҷЁ :)

ејҖжәҗ

жңҖеҗҺдҪҶе№¶йқһжңҖдёҚйҮҚиҰҒзҡ„дёҖзӮ№жҳҜпјҢGo жҳҜдёҖдёӘејҖжәҗйЎ№зӣ®гҖӮ дҪ еҸҜд»ҘеҸӮдёҺ Go йЎ№зӣ®е№¶дёәиҜҘйЎ№зӣ®еҒҡеҮәиҙЎзҢ®гҖӮ

дҪҝз”Ё Go жһ„е»әзҡ„зғӯй—Ёдә§е“Ғ

д»ҘдёӢжҳҜдёҖдәӣдҪҝз”Ё Go жһ„е»әзҡ„жөҒиЎҢдә§е“ҒгҖӮ

  • Google дҪҝз”Ё Go ејҖеҸ‘дәҶ Kubernetes usingгҖӮ
  • Docker, дё–з•ҢдёҠжңҖи‘—еҗҚзҡ„е®№еҷЁе№іеҸ°жҳҜдҪҝз”Ё Go ејҖеҸ‘зҡ„
  • Dropbox е·Іе°Ҷе…¶жҖ§иғҪж•Ҹж„ҹз»„д»¶д»Һ Python иҝҒз§»еҲ° GoгҖӮ
  • Infoblox зҡ„ дёӢдёҖд»ЈзҪ‘з»ңдә§е“ҒжҳҜдҪҝз”Ё Go ејҖеҸ‘зҡ„гҖӮ

е®үиЈ…

Go еҸҜд»Ҙе®үиЈ…еңЁ MacпјҢWindows е’Ң Linux иҝҷдёүдёӘе№іеҸ°дёҠгҖӮдҪ еҸҜд»Ҙд»Һhttps://golang.org/dl/ дёӢиҪҪеҜ№еә”е№іеҸ°зҡ„дәҢиҝӣеҲ¶ж–Үд»¶гҖӮ

Mac OS

д»Һ https://golang.org/dl/ дёӢиҪҪ Mac OS е®үиЈ…зЁӢеәҸгҖӮеҸҢеҮ»ејҖе§Ӣе®үиЈ…гҖӮжҢүз…§жҸҗзӨәиҝӣиЎҢж“ҚдҪңпјҢиҝҷдјҡе°Ҷ Golang е®үиЈ…еңЁ /usr/local/go дёӯпјҢе№¶дё”иҝҳе°Ҷж–Үд»¶еӨ№ /usr/local/go/bin ж·»еҠ еҲ° PATH зҺҜеўғеҸҳйҮҸдёӯгҖӮ

Windows

д»Һ https://golang.org/dl/ дёӢиҪҪ MSI е®үиЈ…зЁӢеәҸгҖӮеҸҢеҮ»ејҖе§Ӣе®үиЈ…пјҢз„¶еҗҺжҢүз…§жҸҗзӨәиҝӣиЎҢж“ҚдҪңгҖӮ иҝҷдјҡе°ҶGoе®үиЈ…еңЁдҪҚзҪ® c:\GoпјҢиҝҳе°Ҷзӣ®еҪ• c:\Go\bin ж·»еҠ еҲ°дҪ зҡ„и·Ҝеҫ„зҺҜеўғеҸҳйҮҸдёӯгҖӮ

Linux

д»Һ https://golang.org/dl/ дёӢиҪҪ tar ж–Үд»¶е№¶е°Ҷе…¶и§ЈеҺӢзј©еҲ° /usr/localгҖӮ

е°Ҷ /usr/local/go/bin ж·»еҠ еҲ° PATH зҺҜеўғеҸҳйҮҸдёӯгҖӮиҝҷе°ҶеңЁ Linux дёӯе®үиЈ… GoгҖӮ

йӘҢиҜҒе®үиЈ…

иҰҒйӘҢиҜҒ Go е®үиЈ…жҳҜеҗҰжҲҗеҠҹпјҢиҜ·еңЁз»Ҳз«Ҝдёӯй”®е…Ҙе‘Ҫд»Ө go versionпјҢе®ғе°Ҷиҫ“еҮәе·Іе®үиЈ…зҡ„ Go зүҲжң¬гҖӮиҝҷжҳҜжҲ‘з»Ҳз«Ҝзҡ„иҫ“еҮәгҖӮ

еңЁзј–еҶҷжң¬ж•ҷзЁӢж—¶пјҢ1.16 жҳҜ Go зҡ„жңҖж–°зүҲжң¬гҖӮиҝҷиҜҒжҳҺ Go е·ІжҲҗеҠҹе®үиЈ…гҖӮеңЁдёӢдёҖдёӘж•ҷзЁӢдёӯпјҢжҲ‘д»¬е°ҶеңЁ Go дёӯзј–еҶҷз¬¬дёҖдёӘ Hello World зЁӢеәҸ :)
